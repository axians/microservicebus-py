Have you thought about the long term?

The challenges that comes with todays IoT-solutions are the well-known silo-effects, on-boarding of new devices, data- and connectivity plan and the so important patching and updates of devices. We see lock-in effects of hardware, IoT-hub and protocols that affect the cost of your IoT-solution.

With microServiceBus.com as your device management layer all the above challenges are mitigated. More features are available out of the box all designed to let you focus on the value that your IoT-solutions provides to you and your customers.

Intel® NUC Rugged

Intel® NUC Rugged is a PC kit that’s small on size but not on toughness or reliability. It features a fanless, ventless design to minimize particle intrusion, so it’s built to survive environments most PCs can’t—from a busy factory floor to a dusty warehouse. Given its modularity, it is designed to work with a range of Intel® NUC Compute Elements ranging from Intel® Celeron® processor all the way up to Intel® Core™ i7 vPro® processor

microServiceBus.com node agents run on Intel® NUC and is a perfect choice for edge computing such as video processing or machine learning.

Supported Device Platforms and frameworks

Node.js® is a runtime built on Chrome's V8 JavaScript engine. Node.js' package ecosystem, npm, is the largest ecosystem of open source libraries in the world.

Gateways, devices or servers running operating systems such as Linux or Windows can run using node.js. Node.js and JavaScript are arguably the most productive developing platform with low learning curve and a massive community of resources.

The Yocto Project (YP) is an open source collaboration project that helps developers create custom Linux-based systems regardless of the hardware architecture.

The project provides a flexible set of tools and a space where embedded developers worldwide can share technologies, software stacks, configurations, and best practices that can be used to create tailored Linux img for embedded and IoT devices, or anywhere a customized Linux OS is needed.

The ARM mbed IoT Device Platform is one of two supported platforms, and is suitable for small microcontrollers developed using C/C++

ARM mbed provides the operating system, cloud services, tools and developer ecosystem to make the creation and deployment of commercial, standards-based IoT solutions possible at scale.

Docker® is an open platform for developers and sysadmins to build, ship, and run distributed applications, whether on laptops, data centers, in the cloud. or on devices.

Docker containers uses resource isolation features of the OS kernel to isolate application's view of the operating environment, including process trees, network, user IDs and mounted file systems, while the kernel's cgroups provide resource limiting, including the CPU, memory, block I/O, and network

Snaps are designed to be secure, sandboxed, containerised applications isolated from the underlying system and from other applications. Snaps allow the safe installation of apps from any vendor on mission critical devices and desktops.

microServiceBus.com® is available as an Ubuntu Snap, which can easily be deployed and updated.

Supported IoT Platforms

Microsoft Azure is a massive growing collection of integrated IaaS and PaaS cloud services for analytics, computing, database, mobile, networking, storage, web and of course IoT.

microServiceBus.com® extends Azure IoT offerings with device- and application management along with On-Air update support for distributing software updates to devices and much more.

AWS has built IoT specific services, such as AWS IoT. They help you collect and send data to the cloud, make it easy to load and analyze that information, and provide the ability to manage your devices, so you can focus on developing applications that fit your needs.

microServiceBus.com® extends AWS IoT offerings with device- and application management along with On-Air update support for distributing software updates to devices and much more.

Google Cloud Internet of Things (IoT) Core is a managed service for securely connecting and managing IoT devices, from a few to millions. Ingest data from connected devices and build rich applications that integrate with the other big data services of Google Cloud Platform.

microServiceBus.com® extends Google IoT Core offerings with device- and application management along with On-Air update support for distributing software updates to devices and much more.

IBM Watson is a cloud platform as a service (PaaS) developed by IBM. As part of IBM Watson, IBM Watson IoT lets you communicate with and consume data from connected devices and gateways. For more information about IBM Watson IoT

microServiceBus.com® extends IBM Watson IoT offerings with device- and application management along with Over-Air update support for distributing software updates to devices and much more.

ProptechOS is an innovation platform which collects and harmonizes real-time data from building management systems, BIM, business and IoT devices. For more information about ProptechOS

ProptechOS is built on RealEstateCore allowing Property owners to describe the data of interaction within the buildings that they operate – as well as the management, storage, and sharing of this data. . For more information about RealEstateCore

Eclipse Mosquitto is an open source (EPL/EDL licensed) message broker that implements the MQTT protocol versions 5.0, 3.1.1 and 3.1. Mosquitto works both in the cloud and on-premises. For more information about Eclipse Mosquitto

Eclipse Mosquitto is a light-weight alternative as an IoT Hub, but could also serve as a on-premise ingestion system when microServiceBus.com® is installed within your data centre.

Our Enterprise offering is fully featured and extended to support medium sized organizations. Enterprise edition comes integrated with Cisco Jasper (SIM-card management) to drastically simplifying the on-boarding process.

Private

Custom domain

€ 1.500 / month

If the multitenant options are not for you, and you require an isolated site, the Private Edition should be your first options.

Our Private offering is hosted in Microsoft Azure through our subscription dedicated to you in a location of your choice. All data will remain isolated from all other organizations.

Your site with all its related artefacts and dependencies will follow the lifecycle of microServiceBus.com with two planned releases per month. This offer comes with both Stage and Production site.

Self-hosted

Custom domain

Our Self-hosted offering is identical to our Private offering but hosted in your own Microsoft Azure subscription. If you already have an Microsoft Azure subscription and want to manage the scale of your platform yourself, this might be the best fit.
